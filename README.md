# «AirKey: Your Offline Voice Assistant for Seamless Typing and Automation»
AirKey — голосовой ввод текста с поддержкой макросов. Работает полностью офлайн. Автоматически копирует надиктованный текст в буфер обмена. Есть функция автозапуска от имени администратора для работы в любых окнах. Идеально подходит для быстрой печати и автоматизации рутины.


AirKey — voice-to-text input with macro support. Works completely offline. Automatically copies dictated text to the clipboard. Includes an "as administrator" autostart feature for compatibility with any window. Perfect for fast typing and routine automation.
