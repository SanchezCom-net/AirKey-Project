# «AirKey: Your Offline Voice Assistant for Seamless Typing and Automation»
AirKey — голосовой ввод текста с поддержкой макросов. Работает полностью офлайн. Автоматически копирует надиктованный текст в буфер обмена. Есть функция автозапуска от имени администратора для работы в любых окнах. Идеально подходит для быстрой печати и автоматизации рутины.

📌 Важные правила при работе с программой  
Путь установки (Только латиница)  
При выборе папки для установки нельзя использовать русский язык (кириллицу).  
❌ Неправильно: C:\Программы\Название  
✅ Правильно: C:\Programs\Name  
Это необходимо для стабильной работы всех функций программы.  
▶️Запуск программы (Фоновый режим)  
У программы нет главного окна, которое открывается сразу. После запуска она работает в фоновом режиме. Ищите значок   программы в системном трее (в правом нижнем углу экрана, рядом с часами).  
Если значка не видно, нажмите на маленькую стрелочку «Отображать скрытые значки».  


AirKey — voice-to-text input with macro support. Works completely offline. Automatically copies dictated text to the clipboard. Includes an "as administrator" autostart feature for compatibility with any window. Perfect for fast typing and routine automation.

📌 Important Rules for Program Installation and Use  
Installation Path (Latin Characters Only)  
When choosing an installation folder, do not use Cyrillic or any non-Latin characters.  
❌ Incorrect: C:\Программы\Name  
✅ Correct: C:\Programs\Name  
This is required to ensure the stability of all program features.  
▶️Launching the Program (Background Mode)  
The program does not have a main window that pops up upon launch. It runs in the background. Look for the program icon in the System Tray (lower right corner of the screen, next to the clock).  
If you don't see the icon, click the small arrow to "Show hidden icons."
